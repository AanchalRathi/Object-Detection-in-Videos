# 🚀 Object Detection in Videos using YOLOv8

## 📌 Overview
This project demonstrates **object detection in videos using deep learning**.  
It uses the **YOLOv8 (You Only Look Once)** model to detect objects such as people, cars, and other common items in a video.

The system processes the video frame-by-frame and draws **bounding boxes with labels and confidence scores**.

---

## 🎯 Features
- Real-time object detection  
- Uses YOLOv8 (latest model)  
- Works on video input  
- Generates output video with detections  
- Simple and easy implementation  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- Ultralytics YOLOv8  
- Deep Learning (CNN)

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/AanchalRathi/object-detection-yolov8.git
cd object-detection-yolov8
```
2. Install dependencies:
pip install -r requirements.txt

▶️ How to Run
python detect.py
